# 🤖 Automação de Cadastro de Produtos

Projeto de automação desenvolvido em Python utilizando PyAutoGUI e Pandas.

## 🚀 Funcionalidades

- Abertura automática do navegador
- Login automático no sistema
- Leitura de base de dados CSV
- Cadastro automático de produtos

## 🛠 Tecnologias utilizadas

- Python 3.14
- PyAutoGUI
- Pandas

## 📂 Estrutura

- codigo.py → Script principal
- produtos.csv → Base de dados utilizada

## ⚠️ Observação

As coordenadas da tela podem variar dependendo da resolução do monitor.
